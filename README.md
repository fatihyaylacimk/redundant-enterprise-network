# Redundant Enterprise Network Design

This project demonstrates a redundant enterprise network designed and simulated using Cisco Packet Tracer.

## Features
- Multi-router topology with redundant links
- Dynamic routing (OSPF / EIGRP)
- Layered network architecture (Access – Distribution – Core)
- Spanning Tree Protocol (STP)
- IP addressing and subnetting
- Network failover and connectivity testing

## Tools
- Cisco Packet Tracer
